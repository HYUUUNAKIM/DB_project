# DB_project
데이터베이스 프로젝트 주제는 학원에서 사용하는 학생 관리 프로그램입니다.  
주제를 선정하게 된 이유는 제가 학원에서 아르바이트하는 동안 원비 관리, 학생 정보 관리, 수업 관리 등 다양한 엑셀을 접해야 했습니다. 분산되어 있는 정보를 한 곳에 모으면 일의 효율이 올라갈 수 있을 것이라는 생각이 들었기 때문에 학생 관리 프로그램을 제작하기로 하였습니다.
