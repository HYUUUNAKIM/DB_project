# DB_project

● 10주차 과제) 요구사항 명세서 작성
1. 학생으로 등록하려면 학생번호, 학교, 학년, 학생전화번호, 학부모전화번호, 주소, 수업을 입력해야한다.
2. 학생은 학생번호로 식별한다.
3. 강의에 대한 수업이름, 강사이름, 교재이름, 수업시간을 유지해야한다.
4. 강의는 수업이름으로 식별한다.
5. 학생과 강의는 수강관계
6. 원비는 학년과 금액을 포함한다.
7. 학생과 원비는 납입관계
8. 납입관계 속성 - 결제날짜, 결제금액, 결제사유, 결제수단
